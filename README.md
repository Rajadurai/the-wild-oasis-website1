### THE WILD OASIS WEBSITE

<h2>Project Requirements</h2>
<ul>
  <li>Users of the app are potential guests and actual guests.</li>
  <li>ABOUT: </li>
  <li>Guest should be able to learn all about the Oasis Hotel.</li>
  <li>CABIN:</li>
  <li>Guest should be able to get information about each cabin and see booked dates.</li>
  <li>Guest shoulb be able to filter cabins by their maximum guest capacity.</li>
  <li>RESERVATION</li>
  <li>Guest should be able to reserve a cabin for a certain data range.</li>
  <li>Reservation are not a paid online payment will be made at the property upon arrival. Therefore, new reservation should be set to "unconfirmed". (booking but not yet checked in)</li>
  <li>Guest should be able to view all their [ast and future reservation.</li>
  <li>Guest should be able to update or delete the reservation.</li>
  <li>AUTHENTICATION</li>
  <li>Guest need to signup and login before they can reserve a cabin and perform any operation.</li>
  <li>On signup, each guest should get a profile in the DB.</li>
  <li>PROFILE</li>
  <li>Guest should be able to set and update basic data about their profile to make check-in at the hotel faster.</li>
</ul>
